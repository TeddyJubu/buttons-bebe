# Sizing and Fit Guidance

## Core rule: Do not guess

**Never** guess sizing, fit, or sleeve length if the information is not confirmed from one of these sources:
- Product page
- Product title or description
- Vendor/brand sizing notes
- Previous staff answer
- Saved product memory or internal notes

If the information is not available from one of these sources → **escalate to a human**.

---

## How an item runs (true to size, small, roomy)

**If known:**
> Hi! This item generally runs [true to size / small / roomy]. Fit can vary by child, but based on what we know about this style, we would recommend [recommendation].

**If unknown — escalate:**
> Hi! We can try to check this for you. Fit can vary by brand and style, and we don't want to guess if we don't have exact notes on this item. We'll check and get back to you with the best information we can.

---

## Sizing for multiple items

Check each item individually against product data and saved notes. If info is available for some but not all → answer what is known and escalate for the rest.

**If info is not immediately available:**
> Hi! We'd be happy to try to help with sizing. Sizing can vary by brand and style, and we don't always have exact sizing details for every item. We'll check and do our best to guide you based on the information we have.

---

## Measurements

Measurements usually require human or warehouse help unless already saved on file.

**Agent steps:**
1. Check if measurements are saved.
2. If saved → provide them.
3. If not saved → escalate to human or warehouse.
4. Do not promise immediate timing.

**Customer response:**
> Hi! We can try to get measurements for you. Please note that measurements may take some time, since someone has to check the item when they are available. If we already have the measurements on file, we'll send them over.

---

## Sleeve length (long sleeve / short sleeve)

Answer only if confirmed from product page, title, description, images, or saved notes.

**If known:**
> Hi! This item is [long sleeve / short sleeve].

**If unknown — escalate:**
> Hi! We're checking that for you and will get back to you as soon as we can.
