# Damaged or Wrong Items

Both situations require photos and human/warehouse involvement. Escalate after initial response.

---

## Customer received the wrong item

**Policy:** The correct item may still be physically available even if inventory does not show extra stock — because the wrong item was shipped out.

**Agent steps:**
1. Apologize.
2. Ask for a photo of the item received, including tag/label if possible.
3. Compare received item to order.
4. Notify warehouse to locate and ship correct item.
5. Determine whether wrong item needs to be returned.
6. Do not rely only on inventory count — always check with warehouse.

**Customer response:**
> Hi! We're so sorry about that. We'll help get this corrected. Please send us a photo of the item you received, including the tag/label if possible, so we can confirm what was sent. Once confirmed, we'll check with the warehouse and work on getting the correct item sent out.

**Escalate to warehouse after receiving photos.**

---

## Customer received a damaged item

**Policy:** Photos are required. Damage must be reported to the vendor/company. Check if replacement is available.

**Agent steps:**
1. Apologize.
2. Ask for clear photos of the damage.
3. Ask for photo of tag/label if needed.
4. Check if a replacement is available.
5. If replacement available → notify warehouse.
6. Email vendor/company with photos and order details.
7. If no replacement → follow refund/return/store-credit policy.

**Customer response:**
> Hi! We're so sorry the item arrived damaged. Please send us clear photos of the damage, along with a photo of the item tag/label if possible. We'll pass this along to the company and check what can be done. If we have another one available, we'll also check whether we can send a replacement.

**Escalate to warehouse and vendor after receiving photos.**

---

## Item looks different from the website photo (color / photo discrepancy)

**Tone first:** Acknowledge the customer's concern warmly. **Do NOT lead with "all our images are taken professionally"** — it reads as dismissive. Open with empathy, then gently explain that lighting and screens can affect how colors appear.

**Determine which situation it is:**
- **Materially different / not as described** (e.g., listed as light denim, received dark denim; or the listing had wrong details like sleeve length or features) → treated as a **store error** → free return label + refund. Follow the wrong-item/returns flow.
- **Subjective color perception** (customer expected a slightly different shade) → a return may be offered if still within the return window, but the return shipping is at the customer's expense. If the item is final sale, no return; during sale season, in-warehouse exchange only.

**Customer response:**
> Hi! We're sorry to hear that. Our photos are taken in professional lighting, which can affect how colors look on different screens — so we'd love to help figure out the best option for you. Could you send a photo of the item you received? If it's materially different from how it was listed, we'll take care of a return for you right away.
