# Refund Explanation and Refund Error Review

## Two types of refunds

### 1. Refund issued before shipping

Usually means an item in the order was out of stock or unavailable before it shipped.

**Agent steps:**
1. Locate the order.
2. Check when the refund was issued (before shipment).
3. Identify which item was unavailable.

**Customer response:**
> Hi! We checked your order. The refund was issued because [item name] was out of stock/unavailable before your order shipped, so you were refunded for that item.

---

### 2. Refund connected to a return

**Do not assume a return-related refund is correct.** Review carefully before confirming.

**Agent steps — review checklist:**
- Which item was refunded?
- Is the refund amount correct?
- Was it refunded to original payment method or issued as store credit?
- Is the return within the refund window?
- Were any items rejected?
- Were restocking or return fees applied?

If anything looks incorrect → escalate to a human before responding.

**Customer response while reviewing:**
> Hi! We'll look into this for you. Since this refund was connected to a return, we want to review the return details carefully to make sure everything was processed correctly. We'll check the items, refund amount, and return status and get back to you.

**Customer response during review:**
> Hi! We're checking into this now. We want to make sure the refund was processed correctly, so we're reviewing the order/return details before confirming.
