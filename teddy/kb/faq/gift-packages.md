# Gift Packages Without Sender Note

## Customer intent

Customer received a package from Buttons Bebe as a gift and wants to know who sent it.

## Agent steps

1. Locate the order by recipient name, address, or tracking number.
2. Check for a gift note on the order.
3. If a note exists → share it with the customer.
4. If no note → check purchaser details.
5. Use privacy judgment before revealing purchaser information.
6. Escalate to a human if unsure whether to share purchaser info.

## Customer response

> Hi! We can try to help identify it. We'll check the order to see if there was a gift note or sender information included. If there is a note, we can pass that along. If there wasn't a note, we'll do our best to help based on the order details.

## Privacy note

Use judgment. If revealing the purchaser's name or details feels like a privacy risk (e.g., potential surprise or sensitive situation), escalate to a human rather than sharing automatically.
