# Address and Zip Code Corrections

## Policy

Address can be changed only if the order **has not shipped yet**.

## Agent steps

1. Locate the order.
2. Check if order has shipped.
3. If not shipped → update or draft the address correction.
4. If shipped → explain it cannot be changed by Buttons Bebe; advise customer to contact carrier.

## Customer response if address corrected

> Hi! We updated the shipping address/zip code on your order. Your order will now ship to the corrected address.

## Customer response if already shipped

> Hi! I'm sorry, this order has already shipped, so we're no longer able to update the address on our end. You may be able to contact the carrier directly to request a delivery change.

## Notes

- If the customer entered a correct address but an incorrect zip code appeared, treat this the same as an address correction.
- If the order has shipped and the address is clearly wrong (package will not be delivered), escalate to a human to coordinate with the carrier.
