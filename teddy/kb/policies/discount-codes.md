# Discounts and Promotions

## First-time customer discount

**Policy:** Buttons Bebe does **not** offer a first-time customer discount.

**Agent action:** Do not offer a code unless there is an active public promotion.

**Customer response:**
> Hi! Thank you so much for reaching out and welcome to Buttons Bebe. At this time, we do not offer a first-time customer discount. However, we do run sales and promotions from time to time, so the best way to stay updated is through our emails/texts and website. Please let us know if you need help with anything while placing your order!

---

## How to apply a discount code

1. Add items to cart.
2. Proceed to checkout.
3. Find the **"Discount code or gift card"** field.
4. Type or paste the code and click **Apply**.
5. Discount appears before payment is completed.

## Common code issues

**Code says "not valid" or "expired":**
- Codes have limited validity — check the expiry date in the promotion email.
- Codes are case-sensitive — copy and paste instead of typing.
- Some codes apply only to specific products or collections.

**"Minimum order not met":**
- The code requires a minimum cart total. Add more items to qualify.

**Forgot to apply code before completing order:**
- Email support@buttonsbebe.com within 24 hours with the order number and code.
- Handled case by case (and only while the order has not shipped). Since it would mean refunding a difference, surface it to a human rather than promising a refund automatically.

## One code per order

Only one discount code can be applied per order. Codes cannot be combined.

---

## Forgot to apply a code — handling

If a customer forgot to apply a valid code at checkout:

**Agent steps:**
1. Verify the code is **valid and still within its promotion window**, and that the order has **not shipped**.
2. If the code is **expired** or the promotion has ended → do not apply it retroactively.
3. Because applying it after the fact would mean refunding a difference, **surface it to a human** rather than promising the refund automatically.
4. **Watch the threshold:** if a discount brings an item to **20% or more off**, the item becomes **final sale** and can no longer be returned. (1–19% off = store credit only.) See [Final Sale](final-sale.md).

**Customer response (expired):**
> Hi! We're sorry — that promotion has ended, so we're unable to apply it retroactively at this time. Thank you so much for your order!

---

## Item went on sale after purchase — no price adjustment

We do **not** offer price adjustments after an order is placed. If a customer bought at full price and the item later goes on sale, the order stands at the price paid — decline kindly. Do not promise a "price-match refund."

**Customer response:**
> Hi! Thank you for reaching out! We don't offer price adjustments after an order is placed, so we're unable to refund the difference when an item later goes on sale. We're sorry for the disappointment, and we hope you love your purchase!
