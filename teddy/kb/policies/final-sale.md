# Final Sale Exception Policy

## What is final sale?

Items marked **"Final Sale"** at the time of purchase are not eligible for return or exchange by mail after the order has shipped.

## Discount threshold — when an item becomes final sale

The discount amount on an item determines whether it can be returned:

| Discount on the item | Return eligibility |
|---|---|
| **20% or more off** | **Final sale** — no return or exchange by mail |
| **1–19% off** | **Store credit only** (no cash refund) |
| Full price | Standard return rules apply (see [Returns](returns.md)) |

Note: we do **not** offer price adjustments after an order is placed — if an item later goes on sale, the order stands at the price paid. See [Discounts and Promotions](discount-codes.md).

## If the order has not shipped yet

Size can still be changed if the requested size is available.

**Agent steps:**
1. Locate order.
2. Check if shipped.
3. If not shipped → check requested size availability and change if possible.

**Customer response:**
> Hi! We were able to switch the size for you before the order shipped. Your order is now updated to size [new size].

## If the order has already shipped — wrong size or exchange request

**Policy:** Final sale items are **not eligible** for regular return or exchange by mail.
Any exception must be warehouse-only and approved by staff.

**Warehouse address for in-person exchange only:**
> Buttons Bebe
> 2133 Lakewood Road Unit 104
> Toms River, NJ

The item must be:
- Unworn
- Unused
- With tags attached

**Agent steps:**
1. Locate order.
2. Confirm it has shipped and is final sale.
3. Do not promise an exception.
4. Escalate or request staff approval if exception may be considered.

**Customer response:**
> Hi! Since this item was final sale and the order already shipped, it is not eligible for a regular return or exchange by mail. If we are able to make an exception, it would need to be exchanged in person at our warehouse only. Our warehouse address is: Buttons Bebe, 2133 Lakewood Road Unit 104, Toms River, NJ. The item would need to be unworn, unused, and with tags attached. Please let us know if you'd like us to check whether an in-person exchange exception is possible.

**Note:** Any in-person exception must be approved by staff — escalate this to a human.
