# Order Changes — Pickup, Shipping, and Size Switches

## Switching pickup to shipping

**Policy:** Can be changed even if already processed for pickup, as long as it has not been picked up.

**Agent steps:**
1. Locate order.
2. Remove from pickup workflow/bin.
3. Confirm shipping address.
4. Check whether order qualifies for free shipping (orders over $75).
5. If qualifies → switch to shipping.
6. If does not qualify → send invoice for shipping charge.
7. Notify warehouse to ship the order.

**Customer response if free shipping applies:**
> Hi! No problem, we can switch your order from pickup to shipping. We updated the order to ship to the address on file. You'll receive tracking once it ships.

**Customer response if shipping payment is needed:**
> Hi! We can switch your order from pickup to shipping. Your order does not meet the free-shipping minimum, so there will be a shipping charge. We can send you an invoice for the shipping cost, and once it's paid, we'll have the order shipped out.

---

## Switching shipping to pickup

**Policy:** Can only be changed if the order has **not shipped yet**.

**Agent steps:**
1. Locate order.
2. Check fulfillment/shipping status.
3. If not shipped → edit order to pickup, refund shipping if applicable, send pickup instructions when ready.
4. If already shipped → explain it cannot be changed.

**Customer response if not shipped:**
> Hi! No problem, we switched your order to pickup. You'll receive pickup instructions once the order is ready.

**Customer response if already shipped:**
> Hi! I'm sorry, this order has already shipped, so we're no longer able to switch it to pickup. You should receive tracking updates from the carrier.

---

## Changing size before shipping

**Policy:** Size can be changed only if the order has **not shipped** and the requested size is **available**.

**Agent steps:**
1. Locate order.
2. Check if order has shipped.
3. Check availability of requested size.
4. If available and not shipped → edit the order.
5. If agent cannot edit directly → draft action for approval.
6. Respond after action is completed.

**Customer response if changed:**
> Hi! No problem, we switched the size for you. Your order is now updated to size [new size].

**Customer response if requested size is unavailable:**
> Hi! We checked, but unfortunately the size you wanted is not available. Please let us know if you'd like to keep the original size or choose another available option.

**Customer response if already shipped:**
> Hi! I'm sorry, this order has already shipped, so we're no longer able to change the size in the order.

## Sale season — in-warehouse exchange only

During an active sale season, **mail returns are not accepted and exchanges can only be done in person at the warehouse.** There are no mail exchanges during a sale.

**Agent steps:**
1. Confirm a sale is currently active.
2. Direct the customer to the warehouse for an in-person exchange.
3. Do not issue a mail return label during sale season.

## Exchange for a completely different item

When a customer wants to return one item and get a **different product entirely** (not just a different size of the same item):

- **By mail:** This is handled as a return plus a new purchase. Issue a return label; once the refund or store credit is processed, the customer places a new order for the item they want.
- **In warehouse:** Can be done as a direct in-person swap.
- For lower-value, straightforward cases, store credit may be issued before the return is physically received — use judgment and escalate for higher-value orders.

**Customer response:**
> Hi! For a different item, the easiest way by mail is to return the original for a refund or store credit, and then place a new order for the item you'd like. If you're local, you're also welcome to swap it in person at our warehouse. Let us know which works best!
