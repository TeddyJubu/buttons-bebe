# Package Protection

## What package protection covers

Package protection is an **optional $1.99 add-on at checkout** (it sometimes auto-adds to orders). It covers a package that is **lost, stolen, or misdelivered** on the way to the customer.

| | Coverage |
|---|---|
| **With package protection** | Lost / stolen / misdelivered package → **full refund** to the customer. |
| **Without package protection** | Not covered — the store cannot take responsibility for packages lost or stolen after they leave the warehouse. Help the customer **file a claim with the carrier** instead. |

**Local pickup orders:** Package protection isn't needed and can be removed and refunded on request.

**Customer response (with protection — lost/stolen):**
> Hi! We're so sorry to hear that. Since you have package protection on this order, we've gone ahead and issued you a full refund. We hope your next order goes more smoothly!

**Customer response (without protection — lost/stolen):**
> Hi! We're so sorry about this. Since package protection wasn't on this order, we're unable to cover lost or stolen packages directly, but we can help you file a claim with the carrier. We'd also recommend checking with neighbors or your building office in case it was set aside for you.

See [Order Tracking](../faq/order-tracking.md) for the full "delivered but not received" flow.

## Remove or refund package protection

If the customer was charged for package protection and wants it removed or refunded, complete the action if possible.

## Agent steps

1. Locate the order.
2. If order is identified, **do not ask for order number**.
3. Cancel or refund the package protection if permitted.
4. If unable to complete, draft for staff approval.
5. Respond **after** action is completed.

## Customer response if completed

> Hi! We went ahead and removed/refunded the package protection for you. You should see the refund back to your original payment method once it is processed by your bank/payment provider.

## Customer response if order cannot be identified

> Hi! We can help with that. Can you please send your order number so we can pull up the order?
