## Return refund window by days since delivery

The day of delivery counts as **Day 1**, per the confirmed
`return-and-exchange-policy.md`:

- **Within 7 days of delivery** (the return package scanned in by the carrier within
  7 days) → cash refund to the original payment method.
- **After the 7-day window** → eligible returns may only qualify for **store credit**,
  not a cash refund (the return portal may auto-issue a gift card; an agent can
  override to a cash refund only if the return was genuinely scanned within 7 days —
  verify the delivery and scan dates first).

Items must come back clean, unworn, unwashed, with all original tags and
accessories. The agent may explain these timings, but the refund/store-credit money
action is escalation-only.

## Baby-gift return window — 21 days, store credit only

Mined practice (confirm the exact window with the owner): baby gifts have a longer
return window of about **21 days from delivery**, and gift
returns are issued as **store credit only** (not a cash refund). Gift returns are
kept private — the sender is not notified. Look up the original order by the invoice
number on the packing slip. See `gifts-and-gift-wrapping.md`.

## Final-sale threshold — 20% or more off

An item discounted **20% or more** is **final sale**: no return or exchange by
mail. During a sale, an in-warehouse exchange may still be possible (see
`sale-season-and-pickup.md`). Note we do **not** offer price adjustments after
purchase (see `promo-codes-and-price-match.md`).

## Items at 1–19% off — store credit only

An item discounted between **1% and 19%** off is returnable for **store credit
only**, not a cash refund. This is below the 20% final-sale threshold, so the item
can still come back, but the refund type is restricted to store credit. Confirm the
discount level on the order before quoting cash vs. store credit.

## System auto-issued store credit — override

The return portal automatically issues store credit (a gift card) when a return is
scanned after the 7-day cash window or when certain conditions are met. If a
customer says they got store credit but were within 7 days: verify the delivery
date and the return scan date. If genuinely within 7 days, the store credit can be
overridden to a cash refund (deactivate the gift card, then issue the cash refund).
If the return was scanned after the 7-day window, store credit is correct — explain
the window. The money override itself is a human action; refunds are escalation-only.
