## Do-not-guess sizing rule (confirmed behavior)

Sizing, how an item runs, measurements, fabric/material, and sleeve length are
**product-specific questions the agent must not guess**. The agent may answer only
if reliable info exists (product page/title/description, vendor data, previous staff
answer, saved product memory, internal notes). If it is not available, **escalate to
a human**. This is confirmed in `agent-core-rules.md` and the confirmed intents
`../intents/intent-04-sizing-help-multiple-items.md`,
`../intents/intent-05-how-does-item-run.md`,
`../intents/intent-09-measurements.md`,
`../intents/intent-11-fabric-material.md`, and
`../intents/intent-13-sleeve-length.md`.

## We sell baby & children's clothing by age/size

Buttons Bebe carries baby and children's apparel from multiple brands. Because we
resell many brands, **fit is not uniform across the store** — a given brand may run
small or large relative to another.

## When a customer asks "what size should I get?"

Do not guess specific measurements. Safe, conservative guidance:
- Ask for the child's age and, if available, current size in a brand they own.
- Note that sizing varies by brand and some items run bigger or smaller than
  expected.
- If reliable brand-specific guidance exists, use it; otherwise escalate rather
  than guessing, and explain our exchange/swap options
  (see `return-and-exchange-policy.md` and `order-changes-and-cancellations.md`).

## "It runs big / small" after delivery

If an item runs noticeably different from expected:
- For an **unshipped** order, a size swap is usually easy (see order-changes).
- For a **delivered sale item**, it is generally final sale; offer a warehouse
  exchange where possible rather than a mail return.
- If two items appear **mis-sized versus each other** (a possible manufacturing
  inconsistency), treat it as a potential defect — gather details/photos and
  check with the vendor before promising anything (see `warranty-and-defects.md`).

## Never do

Never state a specific measurement, inseam, or chest width unless it comes from a
confirmed owner-provided chart. Inventing numbers risks a wrong order and an
avoidable return.
