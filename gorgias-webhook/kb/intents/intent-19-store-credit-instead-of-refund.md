## Policy — refund window vs store credit

Customer expected refund but received store credit.

Refund is available only within the refund window. If customer held item and return was scanned outside the refund window, it becomes store credit. (The refund window is 7 days after delivery — the return must be scanned by the carrier within 7 days; see Intent 20.)

## Agent action

Locate order.
Check delivery date.
Check return scan date.
Check refund window.
Explain whether return was outside refund window.

## Customer response

Hi! Returns are eligible for a refund only within the refund window.
If the item was held onto past the refund window, the return changes to store credit instead of a refund.
We can check your order and confirm the timing for you.
