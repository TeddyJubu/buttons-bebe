## Policy — how an item runs

Customer asks if an item runs true to size, small, or large.

Do not guess. Answer only from reliable product or staff history.

## Agent action

Check product data.
Check saved fit notes.
Check previous customer-service answers.
If known, answer.
If unknown, escalate.

## Customer response if unknown

Hi! We can try to check this for you.
Fit can vary by brand and style, and we don’t want to guess if we don’t have exact notes on this item. We’ll check and get back to you with the best information we can.

## Customer response if known

Hi! This item generally runs [true to size / small / roomy].
Fit can vary by child, but based on what we know about this style, we would recommend [recommendation].
