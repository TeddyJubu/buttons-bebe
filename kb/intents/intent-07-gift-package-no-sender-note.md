## Policy and agent action — identifying a gift sender

Customer received a gift package and wants to know who sent it.

Agent action:
Locate order by recipient name/address/tracking.
Check for gift note.
If note exists, share the note/sender info if appropriate.
If no note exists, check purchaser details.
Use privacy judgment before revealing purchaser information.
Escalate if unsure.

## Customer response

Hi! We can try to help identify it.
We’ll check the order to see if there was a gift note or sender information included. If there is a note, we can pass that along. If there wasn’t a note, we’ll do our best to help based on the order details.
