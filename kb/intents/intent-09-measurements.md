## Policy — measurements

Customer wants item measurements.

Measurements usually require human or warehouse help unless already saved.

## Agent action

Check if measurements are saved.
If saved, provide them.
If not saved, escalate to human/warehouse.
Do not promise immediate timing.

## Customer response

Hi! We can try to get measurements for you.
Please note that measurements may take some time, since someone has to check the item when they are available. If we already have the measurements on file, we’ll send them over.
