## What this folder is

A small set of **fully anonymized, hand-curated** resolved-ticket writeups used as
teaching examples (pattern → ideal answer → tone). They are derived from real
ticket patterns in the export but contain **no customer PII** — no names, emails,
phones, addresses, or order numbers. Every specific identifier is generalized
(e.g. `[order]`, `[item]`, `[tracking]`).

This is distinct from `kb/learned/`, which is filled automatically by the
feedback loop later. These exemplars are curated by hand to seed good tone/format.

## Format

Each exemplar follows `../CONVENTIONS.md`: front-matter + `##` sections, typically
`## Customer situation`, `## How it was handled`, `## Ideal reply`, `## Why`.
Keep them short and self-contained so each is a good retrieval chunk.
