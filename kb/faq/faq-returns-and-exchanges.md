## How long is my order refundable?

An order is refundable for **7 days after delivery**. To qualify for a refund, the
return must be **scanned by the carrier within 7 days** of delivery. After that
window, eligible returns may only qualify for **store credit**. See
`../intents/intent-20-how-long-refundable.md`.

## Why did I get store credit instead of a refund?

Refunds are available only within the 7-day refund window. If the item was held past
that window and the return was scanned by the carrier outside it, the return changes
to **store credit** instead of a refund. We can check the delivery date and scan
date to confirm the timing. See `../intents/intent-19-store-credit-instead-of-refund.md`.

## Can I return / exchange a final sale item that already shipped?

Final sale items that have already shipped are **not eligible for a regular return
or exchange by mail**. Any exception would be a **warehouse-only, in-person
exchange** that must be approved, with the item unworn, unused, and tags attached.
The warehouse address is **Buttons Bebe, 2133 Lakewood Road, Unit 104, Toms River,
NJ**. Do not promise an exception — escalate for approval. See
`../intents/intent-12-final-sale-exchange-exception.md`.

## I ordered the wrong size — can I switch it?

If the order **hasn't shipped**, a size swap is possible when the requested size is
available (this is the normal pre-ship change, not an exception). If it already
shipped, the size can no longer be changed in the order. See
`../intents/intent-08-wrong-size-switch-before-shipping.md`.

## I received the wrong or a damaged item

That's our/the vendor's fault, not a buyer return — final-sale rules don't apply.
Apologize and ask for a **photo** (including the tag/label) so we can confirm what
was sent, then work with the warehouse/vendor on a correction or replacement. A
refund connected to a return is escalated to a human. See
`../intents/intent-15-wrong-item-received.md` and
`../intents/intent-16-damaged-item-received.md`.
